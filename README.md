# 📘 Smart Study Planner

A clean and responsive offline study planner built using pure HTML, CSS, and JavaScript.

Generate day-by-day study schedules directly in your browser without any API keys or backend setup.

---

## ✨ Features

- ✅ Fully offline
- ✅ No API key required
- ✅ Responsive mobile-friendly design
- ✅ Add unlimited subjects
- ✅ Difficulty selection
- ✅ Weak areas input
- ✅ Auto-generated study sessions
- ✅ Daily study schedule generation
- ✅ Simple GitHub Pages deployment

## 📥 Installation

### 1. Clone Repository

```bash

```

### 2. Open Folder

```bash
cd study-planner
```

### 3. Run Project

Simply open:

```bash
index.html
```

in your browser.

## 🌐 Deploy on GitHub Pages

### Step 1

Push your project to GitHub.

### Step 2

Go to:

```text
Repository → Settings → Pages
```

### Step 3

Under:

```text
Build and Deployment
```

Choose:

```text
Deploy from branch
```

### Step 4

Select:

```text
Branch: main
Folder: /root
```

### Step 5

Click:

```text
Save
```

Your site will be live at:

```text
https://your-username.github.io/repository-name/
```

## 🧠 How It Works

The app generates schedules locally using JavaScript logic.

It:
- rotates subjects automatically
- balances study sessions
- creates daily schedules
- prioritizes revision
- adapts based on study hours

No server or AI API is required.
